# Functions Practice

## Related Chapter
- Chapter 5: Functions

## Topics Covered
- Defining functions
- Function parameters
- Return values
- Reusing code with functions
- Menu-based programs using functions

## Tasks

### Task 7: Utility Functions
- Implements basic utility functions:
  - Check if a number is even
  - Calculate factorial of a number
  - Calculate the average of a list of numbers
- Demonstrates function calls and return values

### Task 8: Menu-Based Program
- Displays a menu to the user
- Executes different functions based on user choice
- Continues running until the user exits the program

## Objective
The goal of these tasks is to understand how functions help organize code,
reduce repetition, and improve readability in Python programs.
