# Lists and Tuples Practice

## Related Chapter
- Chapter 7: Lists and Tuples

## Topics Covered
- Creating and using lists
- Indexing and slicing
- Iterating over lists
- Basic list operations
- Introduction to tuples

## Tasks

### Task 11: Shopping List
- Creates a shopping list using a list
- Allows the user to:
  - Add items
  - Remove items
  - View the shopping list
- Demonstrates basic list manipulation

### Task 12: Exam Statistics
- Uses a list of exam grades
- Finds and displays:
  - The highest grade
  - The lowest grade
  - The average grade
- Demonstrates iteration and basic calculations with lists

## Objective
The aim of these tasks is to gain practical experience with lists and tuples,
and to understand how ordered collections are used in Python programs.

- `task12_exam_statistics.py`
