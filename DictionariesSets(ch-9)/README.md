# Dictionaries & Sets Practice

## Related Chapter
- Chapter 9: Dictionaries and Sets

## Topics Covered
- Creating and using dictionaries
- Adding, updating, deleting dictionary elements
- Searching in dictionaries
- Iterating over dictionaries
- Using sets to store unique values
- Combining strings, lists, sets, and functions

## Tasks

### Task 15: Phone Book
- A simple phone book application using a dictionary
- Features:
  - Add a new contact
  - Remove an existing contact
  - Search for a contact by name
- Each operation is implemented as a separate function

### Task 16: Unique Words
- Takes a sentence from the user
- Removes punctuation marks
- Converts text to lowercase
- Uses a set to find unique words
- Prints all unique words

## Objective
The objective of these tasks is to practice using dictionaries and sets
together with strings, loops, and functions to solve basic problems.

## Files
- `task15_phonebook.py`
- `task16_unique_words.py`
