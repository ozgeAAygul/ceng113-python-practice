# Strings Practice

## Related Chapter
- Chapter 8: More About Strings

## Topics Covered
- String indexing and slicing
- String methods (`lower`, `upper`, `isdigit`, `isupper`, etc.)
- Iterating over strings
- String comparison
- Basic text processing

## Tasks

### Task 13: Password Validator
- Takes a password input from the user
- Checks whether the password:
  - Is at least 8 characters long
  - Contains at least one digit
  - Contains at least one uppercase letter
- Displays clear error messages for missing requirements
- Uses functions and string methods

### Task 14: Text Analyzer
- Takes a sentence or text from the user
- Calculates:
  - Number of words
  - Character count (excluding spaces)
  - Most frequent word
- Ignores case sensitivity
- Demonstrates string manipulation and dictionaries

## Objective
The purpose of these tasks is to improve understanding of string manipulation
in Python and to practice analyzing text data using loops, functions, and
built-in string methods.
