# Decision Structures Practice

## Related Chapter
- Chapter 3: Decision Structures and Boolean Logic

## Topics Covered
- if / elif / else statements
- Boolean expressions
- Logical operators (and, or, not)
- Comparison operators

## Tasks

### Task 3: Letter Grade Calculator
- Takes an exam average from the user
- Determines the letter grade using decision structures
- Outputs the corresponding letter grade (A, B, C, D, F)

### Task 4: Login Simulation
- Simulates a simple login system
- Checks username and password using conditional statements
- Displays appropriate access messages

## Objective
The purpose of these tasks is to practice decision-making in Python programs using
conditional statements and Boolean logic.
